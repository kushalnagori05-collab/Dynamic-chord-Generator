# DynamicChord 🎸🎹

**Learn, Play, and Master Music**

DynamicChord is a lightweight, responsive web application designed for both guitarists and pianists. It makes learning and practicing music straightforward with tools for song discovery, chord transposition, a comprehensive chord library, and a built-in metronome.

## 🌟 Key Features

* **Song Library:** Search through a collection of popular songs (English and Hindi). View chords, strumming patterns, and arpeggio patterns.
* **Instant Transposition:** Transpose chords in real-time to match your vocal range.
* **Smart Capo Suggestions:** Automatically calculates guitar capo placements to maintain original chord shapes when transposing.
* **Chord Library:** A comprehensive dictionary of over 100+ chords (Major, Minor, 7th, Sus, Dim, Aug, etc.) featuring accurate guitar fret diagrams and piano keyboard views.
* **Built-in Metronome:** An interactive visual and audio metronome with adjustable BPM and pendulum animations to help you keep perfect time.
* **Interactive UI:** Toggle seamlessly between Guitar and Piano views.

## 🛠️ Technology Stack

* **HTML5 / CSS3:** Vanilla CSS for rich, responsive, and dynamic styling.
* **JavaScript / React 18:** Functional components and hooks used for UI state management.
* **Babel (Standalone):** Compiles JSX directly in the browser for a serverless, zero-build setup.
* **Web Audio API:** Generates accurate metronome clicks and plays synthesized chord tones.

## 🚀 Getting Started

Since DynamicChord is built as a static application, you don't need any complex development environment or package installations to run it locally.

1. Clone the repository:
   ```bash
   git clone https://github.com/kushalnagori05-collab/Dynamic-chord-Generator.git
   ```
2. Navigate into the project folder:
   ```bash
   cd Dynamic-chord-Generator
   ```
3. Open `index.html` in your favorite web browser – and you're good to go!

## 💡 How to Use

* **Songs:** Browse or search for a song. Click on it to see the chords. Use the `+` and `-` buttons to transpose keys. 
* **Learn:** Explore different chords by filtering through root notes and chord types. Click into any chord to see fretboard and keyboard diagrams, and hit "Play Chord" to hear it.
* **Practice:** Open the Metronome by clicking the metronome icon, set your desired BPM, and practice your timing.
